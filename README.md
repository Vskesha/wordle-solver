# wordle-solver
CLI program for solving worldle puzzle

## How to use
clone this repo

```
git clone https://github.com/Vskesha/wordle-solver.git
```
Inside root directory of the project run:
```aiignore
python main.py
```
And you will see all available commands:
```aiignore
Available commands: 
help               - Display this help message
start              - Start a new game
quit               - Exit the game
exit               - Exit the game
show N             - Show N random words that fit.
filter CCCCC XXXXX - For filtering words.
                     CCCCC - 5 characters word (case insensitive)
                     XXXXX - where X is one of:
                         0 - letter is not present in the word
                         1 - letter is on wrong position
                         2 - letter is on correct position

>>> 
```
Enjoy simple solving of the puzzle with this app.
